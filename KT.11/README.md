# RAG System - Система анализа технических задания

## Как запустить программу?

### Вариант 1: Готовая сборка
1. Скачайте папку с программой
2. Запустите файл RAGSystem.exe
3. Программа откроется в отдельном окне

### Вариант 2: Из исходного кода
1. Установите Visual Studio 2022 или новее
2. Откройте файл RAGSystem.sln
3. Нажмите кнопку "Запуск" (F5)
4. Программа соберется и запустится

## Как пользоваться программой?

### Шаг 1: Загрузите ТЗ
1. Нажмите кнопку "Загрузить ТЗ"
2. Выберите файл с техническим заданием (.txt)
3. Текст появится в левой части окна

### Шаг 2: Разбейте текст на части
1. Нажмите кнопку "Разбить текст"
2. Программа разделит ТЗ на небольшие блоки
3. В правой части появятся все части с номерами

### Шаг 3: Сохраните векторы
1. Нажмите кнопку "Сохранить"
2. Создастся файл vectors.json
3. Этот файл содержит все разбитые части

### Шаг 4: Ищите информацию
1. Введите вопрос в поле поиска
2. Нажмите "Найти" - найдет похожие части
3. Или "Ответить" - сформирует ответ
